## Dicas para Melhores Resultados

### Midjourney

1. **Aspect Ratio**: Usa `--ar 21:9` para hero, `--ar 4:5` para cards
2. **Version**: Usa `--v 6` para melhor qualidade
3. **Style Raw**: Adiciona `--style raw` para menos "efeitos AI"
4. **Seed**: Guarda seeds bons para consistência

### DALL-E 3

1. **Sê descritivo**: DALL-E responde bem a prompts longos e detalhados
2. **Especifica cores**: Usa códigos hex quando possível
3. **Menciona uso**: "suitable for website hero background"

### Leonardo.ai

1. **Presets**: Usa "Photoreal" para fotografia realista
2. **Negative prompts**: Adiciona o que NÃO queres
3. **Guidance Scale**: 7-9 para mais fidelidade ao prompt
